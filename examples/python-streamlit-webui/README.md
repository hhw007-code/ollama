# Streamlit example
This example uses streamlit to build a chat application front-end from pure Python code.

## Running the Example

1. ensure you have llama3.2 installed 

```bash
ollama pull llama3.2
```

2. Install the Python Requirements.

```bash
pip install -r requirements.txt
```

3. Run the example

```bash
streamlit run streamlitui.py
```
